# My Hotel Website

A simple hotel website project built using HTML and CSS.

## Project Structure

- `index.html` - Home page
- `about.html` - About page
- `menu.html` - Menu page
- `style1.css` - Main stylesheet
- `images/` - Folder containing project images

## Features

- Clean and responsive hotel website design
- Home page with hotel introduction
- About page with hotel details
- Menu page for food items
- Organized file structure

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Clone this repository.
2. Open the `Project-2` folder.
3. Open `index.html` in any web browser.
