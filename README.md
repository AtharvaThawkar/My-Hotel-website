# 👑 Hotel Crown — Hotel & Restaurant Website

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![Bootstrap](https://img.shields.io/badge/Bootstrap_5-7952B3?style=for-the-badge&logo=bootstrap&logoColor=white)
![Responsive](https://img.shields.io/badge/Responsive-Design-06B6D4?style=for-the-badge&logo=google-chrome&logoColor=white)

A fully responsive, multi-page **Hotel & Restaurant website** built using HTML5, CSS3, and Bootstrap 5. Features an elegant dark theme, food menu with pricing, hotel amenities showcase, table booking system, and smooth navigation across multiple pages.

---

## 🌐 Live Demo

> 🔗 **[View Live Demo](https://myhotelproject2.netlify.app/)**

## 📄 Pages

| Page | File | Description |
|---|---|---|
| 🏠 Home | `index.html` | Landing page with hero, gallery & contact |
| 🍽️ Menu | `menu.html` | Full food menu with 9 items & pricing |
| ℹ️ About | `about.html` | Hotel features — 5 Star, Pool, Sea View |

---

## ✨ Features

- 🌑 **Dark Elegant Theme** — Professional dark color scheme (#212529) throughout all pages
- 📱 **Fully Responsive** — Works seamlessly on mobile, tablet & desktop using Bootstrap 5 grid
- 🧭 **Multi-Page Navigation** — 3 linked pages (Home, Menu, About) with consistent navbar
- 🍕 **Food Menu Page** — 9 international dishes (Pizza, Sushi, Tacos, Dosa, Samosa and more) with descriptions & prices
- 📅 **Book Now Modal** — Table/room booking popup form available on every page
- 🏨 **Hotel Features Showcase** — Alternating image-text layout highlighting 5 Star, Infinity Pool, Under Sea View & Sea View Bedrooms
- 📬 **Contact & Enquiry Form** — Full enquiry form with name, email, phone & message fields
- 🔗 **Social Media Links** — Instagram & Facebook footer links
- ⚡ **Smooth Scroll** — CSS smooth scrolling for better user experience
- 🎨 **Custom Favicon** — Hotel Crown logo as browser tab icon

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Edge)
- No installation required — pure frontend project

### Run Locally

1. **Clone the repository**
```bash
git clone https://github.com/AtharvaThawkar/hotel-crown.git
```

2. **Navigate to the project folder**
```bash
cd hotel-crown
```

3. **Open in browser**
```bash
# Simply open index.html in your browser
# Or use Live Server extension in VS Code for best experience
```

## 🎨 Design Highlights

### Dark Elegant Theme
```css
/* Consistent dark theme across all pages */
#First, #Second, #menu, #dd, .jumbotron {
    background-color: #212529;
    color: white;
}
```

### Hero Section
```css
/* Full-viewport hero with background image */
#hero {
    background: url('images/hero-bg.jpg') no-repeat center center/cover;
    height: 60vh;
    display: flex;
    align-items: center;
    justify-content: center;
}
```

### Gold Accent Button
```css
/* Premium gold color for CTA buttons */
.btn-submit {
    background-color: #AD8B3A;
    color: white;
}
```
